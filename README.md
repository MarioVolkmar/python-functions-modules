# Python Functions and Modules

This repository contains Python practice exercises focused on functions, parameters, return values, modules, and basic code organization.

The goal of this repository is to move from writing isolated scripts to writing more reusable, organized, and maintainable Python code.

---

## Main Focus

This repository focuses on:

* Basic functions
* Parameters
* Return values
* Function naming
* Reusable logic
* Lists with functions
* Dictionaries with functions
* String processing with functions
* Basic modular code organization
* Small practice exercises
